<h1> Train Scheduler </h1>

<h2> Logic </h2>
<ul>
  <li> An firebase powered application that simulates a train schedule where the user can create their own train trip which includes: </li>
    <ul>
      <li> Train Name </li>
      <li> Destination (City) </li>
      <li> Arrival Time </li>
      <li> Minutes Away </li>
  </ul>
</ul>

<h2> Technologies </h2>
<ul>
  <li> Javascript </li>
  <li> jQuery </li>
  <li> Firebase </li>
  <li> HTML </li>
  <li> CSS </li>
</ul>
